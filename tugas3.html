Halaman FAQ

1. Apa itu jQuery?
jQuery adalah library JavaScript untuk mempermudah manipulasi DOM.
2. Apa fungsi FAQ?
FAQ membantu menjawab pertanyaan umum pengguna.
3. Apa itu toggle?
Toggle digunakan untuk menampilkan dan menyembunyikan elemen.
4. Apa fungsi hover?
Hover mendeteksi saat mouse berada di atas elemen.
5. Apakah jQuery masih digunakan?
Ya, jQuery masih digunakan pada banyak website lama dan sederhana.
